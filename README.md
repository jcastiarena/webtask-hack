Webtask-hack
=========

A small exercise to test webtasks

## Installation
  Install webtask commandline application:
  
  `npm i -g wt-cli`

  `wt init`

  `wt create search.js`

## Usage

    curl 'https://webtask.it.auth0.com/api/run/wt-<your_id_here>search?webtask_no_cache=1&query=<your_search_term>'
